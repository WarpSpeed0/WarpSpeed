<h1 align="center">Introducing WarpSpeed!</h1>


"But what IS WarpSpeed?"

WarpSpeed is a turbowarp fork I made specifically FOR animators and artists! 

"What's so different between WarpSpeed and Turbowarp?" 

Here are all the features!:

<h2 align="center">--------------------------------------------------------------------</h2>

Layers, Each sprite now gets their own layers. You can copy, delete, hide, and lock layers.

Lasso tool, you can now make precise selections.
(bitmap and vector)

References, you can click on the "reference image" button in the costume editor, there you can import any file that's supported and have that as a window for you to look at during drawing/animating.

You can now choose between ffmpeg and MediaBunny for recording and you can change the video bitrate, making the recording look way better.

Pressure, Costume editor now supports pressure for drawing tablets (Bitmap + vector).

<h2 align="center">--------------------------------------------------------------------</h2>

Upcoming:

Rigs, you can go into "rig" mode in the costume editor (vector only) there you can snap anchor points of objects together an group them as a "character." Your characters will get saved in the backpack and the project

Anchor points, Being able to change the anchor points on groups/objects in vector
