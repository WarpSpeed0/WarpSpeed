<h1 align="center">Introducing WarpSpeed!</h1>


"But what IS WarpSpeed?"

WarpSpeed is a turbowarp fork I made specifically FOR animators and artists! 

"What's so diferent between WarpSpeed and Turbowarp?" 

Well, here's all the features!:

<h2 align="center">--------------------------------------------------------------------</h2>

Each sprite now gets there own layers. You can copy layers, delete layers, hide layers, and lock layers.

Lasso tool, you can now make precise selections.
(bitmap and vector)

References, you can click on the "reference image" button in the costume editor, there you can import any file that's supported and have that as a window for you to look at during drawing/animating.

You can now choose between ffmpeg and MediaBunny for recording and you can change the video bitrate, making the recording look way more better.

Costume editor now supports pressure for drawing tablets (bitmap and vector)

<h2 align="center">--------------------------------------------------------------------</h2>
